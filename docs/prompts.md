# 固定画风提示词

下面的提示词用于生成每一幕的“彩色母图”。它的作用是锁定画风，不负责生成动画；线稿显现和颜色推进由 HyperFrames 完成。

## 固定风格段

把这一段原样放在每一幕提示词的开头，不要随意改写：

~~~
成人故事感的手绘插画，清晰、纤细、连续的深灰黑色钢笔线稿，轮廓和内部结构线都要可辨认，线条有轻微手绘起伏但不能潦草成黑块；低饱和度彩铅轻柔填色，颜色克制、纸张颗粒细腻，局部颜色自然叠加在线稿内部；大面积温暖纸白留白，主体主要位于画面下方45%到55%，上方保留干净安全区；生活化、安静、克制、具有叙事因果的构图；背景只保留一到两个有用的环境锚点；人物比例自然，手、腿、鞋子、衣服边缘完整清楚；适合后续做从左到右的线稿显现和从左到右的渐进着色动画；完整彩色静帧，非黑白草图，非海报排版，非照片。
~~~

## 每幕变量段

把下面的占位内容替换成当前分镜信息，放在固定风格段之后：

~~~
第{幕号}幕，{地点和时间}。
在场人物：{人物1}、{人物2}。
关键动作：{一个明确的动作或反应}。
关键道具：{一个推动情节的道具}。
环境细节：{最多两个背景锚点}。
景别和构图：{远景 / 中景 / 近景 / 道具特写}，动作方向从左向右或符合故事推进方向。
情绪结果：{这一幕结束时观众能看见的变化}。
~~~

## 负面提示词

把这一段放在负面提示词字段；如果供应商没有负面提示词字段，就直接接在主提示词末尾：

~~~
不要照片写实，不要3D渲染，不要动漫厚涂，不要鲜艳霓虹色，不要高饱和渐变，不要复杂建筑，不要人群，不要密集背景，不要满屏雨线，不要大面积纯黑，不要粗重卡通轮廓，不要白色边框，不要白色圆形地面，不要裁掉腿和鞋子，不要缺失手指，不要缺衣服，不要人物漂浮，不要字幕，不要水印，不要 logo，不要分镜拼贴，不要重复人物，不要从上到下出现的动作构图。
~~~

## 参考图锁定角色

使用参考图编辑时，在变量段前加入：

~~~
严格参考输入图中的主角外观、发型、服装轮廓、鞋子形状、线条粗细和纸张质感；保持角色身份和画风一致，只改变本幕的地点、动作、道具和人物关系。不要重新设计主角，不要改变年龄、发型、服装主色和身体比例。
~~~

同一个故事中，建议固定一张角色参考图和一张画风参考图。每幕仍然要生成不同的完整彩色母图，不能用同一张图裁切凑时长。

第1幕，凌晨四点的街角早餐摊。
在场人物：一名清扫街道的工人、一位早餐摊老板。
关键动作：清扫工人推着扫帚从左向右经过摊位，老板抬头看见他。
关键道具：早餐摊、扫帚、保温桶。

景别和构图：下方偏宽的建立镜头，人物和摊位位于下半部，动作方向从左向右。
情绪结果：观众看见两个人已经形成日常相遇的关系。
